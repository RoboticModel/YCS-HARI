ESP32 LED Blinker

Simple code to blink an LED on ESP32 using GPIO pin 2.

Code:

// ESP32 LED Blinker
const int ledPin = 2;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH);
  delay(1000);
  digitalWrite(ledPin, LOW);
  delay(1000);
}
